Documentação Chaplin

Plataforma de Gestão de Tarefas de Manutenção para empresas terceirizadas

🎯 Sobre o Projeto

Chaplin é uma plataforma web desenvolvida como projeto de TCC para empresas terceirizadas de manutenção em hotéis e propriedades de aluguel. A aplicação centraliza a comunicação entre gestores de prédios e equipes de manutenção.

📋 Requisitos

Python 3.8+

Linguagem de programação

MySQL 5.7+

Banco de dados

pip

Gerenciador de pacotes

Git

Controle de versão

🚀 Como Rodar Localmente

1️⃣ Clonar o Repositório

git clone https://github.com/Kayque-Freitas/Chaplin-TCC.git
cd Chaplin-TCC

2️⃣ Criar Ambiente Virtual

Windows:

python -m venv venv
venv\Scripts\activate

Linux/Mac:

python3 -m venv venv
source venv/bin/activate

3️⃣ Instalar Dependências

pip install -r requirements.txt

4️⃣ Configurar Banco de Dados

Criar banco MySQL:

CREATE DATABASE chaplin_db;
CREATE USER 'chaplin_user'@'localhost' IDENTIFIED BY 'sua_senha';
GRANT ALL PRIVILEGES ON chaplin_db.* TO 'chaplin_user'@'localhost';
FLUSH PRIVILEGES;

Criar arquivo `.env`:

cp .env.example .env

5️⃣ Executar Migrações

python manage.py migrate

6️⃣ Criar Superusuário

python manage.py createsuperuser

7️⃣ Rodar o Servidor

python manage.py runserver

Acesse: http://localhost:8000

🔐 Roles de Usuário

👨‍💼 Admin

Gerencia empresa, usuários e relatórios

🏢 Gestor do Prédio

Cria tarefas e valida conclusão

👥 Líder de Equipe

Recebe tarefas e aloca a colaboradores

🔧 Colaborador

Executa tarefas e registra evidências

📝 Fluxo de Tarefas

1

Gestor cria tarefa na plataforma

2

Tarefa é enviada para o Líder (o "tobogã")

3

Líder aloca tarefa ao Colaborador

4

Colaborador executa e registra evidências

5

Gestor valida e finaliza

🛠️ Comandos Úteis

# Criar migrações

python manage.py createsuperuser

# Coletar arquivos estáticos
python manage.py collectstatic

python manage.py shell
