❓ Perguntas Frequentes

Como faço para rodar o projeto localmente? +

Consulte o guia de documentação para instruções passo-a-passo de instalação e configuração.

Qual é a licença do projeto? +

Este é um projeto de TCC (Trabalho de Conclusão de Curso) desenvolvido para fins acadêmicos. Consulte o repositório GitHub para mais detalhes sobre a licença.

Posso usar este projeto em produção? +

Este projeto foi desenvolvido como TCC e pode ser usado como base para projetos comerciais, mas recomenda-se adaptações de segurança e performance para ambiente de produção.

Como contribuir para o projeto? +

Você pode fazer um fork do repositório no GitHub, criar uma branch com suas melhorias e enviar um Pull Request. Consulte o README para mais detalhes.
